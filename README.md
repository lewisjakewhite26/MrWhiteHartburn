# Hartburn — Year 4 Portrait Photography Lesson

Interactive lesson deck (HTML/CSS/JS) for teaching portrait photography on an interactive whiteboard.

## Run locally

```bash
py -m http.server 8765 --bind 127.0.0.1
```

Open http://127.0.0.1:8765/

## Deploy

Works as a static site on Vercel, Netlify, or GitHub Pages. Entry file: `index.html`.
